# SQL Sales Data Analyst Project

##  Project Overview

**SQL Sales Data Analyst** is a MySQL-based data analysis project focused on analyzing sales data and generating business insights.

This project demonstrates SQL skills including data exploration, filtering, aggregation, revenue analysis, customer analysis, and advanced SQL queries.

---

##  Technologies Used

* MySQL
* MySQL Workbench
* SQL
* GitHub

---

##  Database Information

**Database Name**

```sql
sales_Database
```

**Table Name**

```sql
sales_data
```

##  Analysis Performed

### Data Exploration

* Display complete sales records
* Find customers and products
* Filter data based on product and city

### Sales Analysis

* Total orders count
* Total revenue calculation
* Average order value
* Revenue by category
* Revenue by city

### Customer Analysis

* Unique customer count
* Top customers by revenue
* Repeat customer identification
* Customer spending ranking

### Product Analysis

* Product-wise quantity analysis
* Highest revenue products
* Category performance

### Time Analysis

* Monthly revenue analysis
* Date-wise sales performance

##  Business Questions Answered
